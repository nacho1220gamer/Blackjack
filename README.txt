# Blackjack - Proyecto C++

Este proyecto implementa una versión de Blackjack con interfaz gráfica utilizando C++. Está organizado en dos carpetas principales:

## Estructura del Proyecto

- **`FirstGUI/`**: Contiene los archivos fuente del proyecto, incluyendo:
  - Archivos `.cpp` (código fuente)
  - Archivos `.h` (cabeceras)

- **`BlackjackPortable/`**: Contiene el ejecutable del programa listo para usar.

## Cómo ejecutar

1. Ve a la carpeta `BlackjackPortable`.
2. Ejecuta el archivo correspondiente a tu sistema operativo (.exe, .bat).

Asegúrate de tener los permisos adecuados para ejecutar archivos en tu sistema.

## Requisitos de compilación

Si deseas compilar el proyecto desde el código fuente (`FirstGUI`), asegúrate de tener un entorno de desarrollo C++ con soporte para interfaces gráficas, como:
- Visual Studio 2022 (Windows)
